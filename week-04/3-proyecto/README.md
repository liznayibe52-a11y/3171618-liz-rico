# 🚗 Sistema de Parqueadero

## 📌 Descripción del dominio

Este proyecto consiste en una aplicación web desarrollada con React y TypeScript que simula la gestión de un parqueadero.

El sistema permite visualizar los diferentes espacios disponibles dentro del parqueadero, mostrando información como:

* Tipo de espacio (carro, moto, bicicleta)
* Tarifa del servicio
* Estado (disponible u ocupado)
* Calificación del espacio

Además, el usuario puede:

* Buscar espacios en tiempo real
* Filtrar por tipo de vehículo
* Mostrar solo espacios disponibles
* Ordenar por nombre, tarifa o calificación

Este proyecto implementa conceptos como hooks, manejo de estado, renderizado condicional y optimización con debounce.

## ⚙️ Instrucciones de ejecución

Sigue estos pasos para ejecutar el proyecto:

1. Clonar el repositorio o copiar el proyecto

2. Instalar dependencias:

```bash
pnpm install
```

3. Ejecutar el proyecto:

```bash
pnpm dev
```

4. Abrir en el navegador:

```
http://localhost:5173
```

# 🚗 Sistema de Gestión de Vehículos en Parqueadero

Aplicación web desarrollada con **React y TypeScript** que permite gestionar el ingreso de vehículos a un parqueadero.  
El sistema permite **agregar, editar y eliminar vehículos**, mostrando la información en una lista dinámica.

---

# 📋 Descripción del Proyecto

Este proyecto consiste en una aplicación que administra el registro de vehículos que ingresan a un parqueadero.  
Permite registrar información como:

- Placa del vehículo
- Tipo de vehículo (carro o moto)
- Hora de entrada
- Estado del vehículo (en parqueadero o fuera)


---

# ⚙️ Tecnologías Utilizadas

- React
- TypeScript
- Vite
- CSS
- JavaScript 

---

# 🧠 Conceptos Aplicados

Durante el desarrollo del proyecto se aplicaron los siguientes conceptos:

- Componentes funcionales en React
- Uso de **useState** para manejo de estado
- Uso de **useEffect**
- Interfaces en TypeScript
- Props tipadas
- Formularios controlados
- Renderizado dinámico con **map()**
- Eliminación de elementos con **filter()**
- Actualización de datos usando **spread operator**
- Patrón CRUD (Create, Read, Update, Delete)

---

# 🚀 Funcionalidades

### ➕ Agregar Vehículo
Permite registrar un nuevo vehículo ingresando:

- Placa
- Tipo de vehículo
- Hora de entrada
- Estado

### ✏️ Editar Vehículo
Permite modificar los datos de un vehículo existente.

### ❌ Eliminar Vehículo
Permite eliminar un vehículo del sistema.

### 📄 Listar Vehículos
Muestra todos los vehículos registrados en forma de tarjetas.

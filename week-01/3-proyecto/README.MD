## 📌 Descripción
Este proyecto implementa la gestión básica de entidades utilizando TypeScript.
Se crean funciones para:

- Crear entidades
- Listar entidades
- Filtrar entidades por estado y categoria

## 🛠 Tecnologías
- TypeScript
- Node.js

## 📂 Estructura
Productos:
- Interface Vehicle
- Type vehicleType
- Función createVehicle()
- Función listVehicle()
- Función filterByVehicleType()

Clientes:
- Interface Shift
- Type shiftStatus
- Función createShift()
- Función listShift()
- Función filterByStatus()

## 🚀 Ejecución

1. Instalar dependencias:
   npm install

2. Ejecutar el proyecto:
   npm start
